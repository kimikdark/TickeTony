Perguntas Frequentes

Encontre respostas para as perguntas mais comuns sobre o TickeTony.

O que é o TickeTony?

O TickeTony é uma plataforma open source de gestão de tickets e tarefas, desenhada para equipas ágeis. Ajuda a centralizar a comunicação, organizar o trabalho e otimizar processos.

Como posso começar a usar?

Atualmente, o TickeTony está em fase de desenvolvimento. Em breve, disponibilizaremos um link para demonstração e instruções detalhadas para começar a utilizar a plataforma.

O TickeTony é gratuito?

Sim, o TickeTony é uma plataforma open source e será totalmente gratuita para utilizar. O nosso objetivo é fornecer uma ferramenta acessível e eficiente para todas as equipas.

Posso contribuir para o projeto?

Absolutamente! Acreditamos no poder da comunidade. Em breve, disponibilizaremos o repositório no GitHub e guias para que programadores, designers e outros contribuam para o TickeTony.

Qual o prazo de resposta?

A nossa equipa esforça-se para responder a todas as mensagens submetidas via formulário num prazo máximo de 48 horas úteis. Em casos de grande volume, pode demorar um pouco mais, mas garantimos que todas as questões serão atendidas.
